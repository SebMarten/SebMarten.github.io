Linear Regression in Excel

For one of my university classes, I was tasked to conduct some very simplified linear regression on imaginary demographic student data.

In this university assignment, our class was asked to complete a Google Form for the demographics and personality type of an imaginary friend. The assignment involved conducting linear regressions to impute five missing values in the student data file using the Excel ‘Data Analysis’ add-in. After then merging the demographics and personality sheets using ‘V-Look-Up’, we were tasked to tell an intersting story about this small data set.
